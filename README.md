# Stock Screener UI Design

This is a React + Vite application for the Stock Screener UI. The original design is available at https://www.figma.com/design/C855gkWLeVMjLUgiXqCSUz/Stock-Screener-UI-Design.

### Prerequisites
- Node.js (v16 or higher) - [Download here](https://nodejs.org/)

### First Time Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```
   (or `npm i` - both commands do the same thing)

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - The app will automatically open at `http://localhost:3000`
   - Or manually navigate to that URL

### Daily Development

Just run:
```bash
npm run dev
```

That's it! The server will start and your browser will refresh automatically when you make changes.
